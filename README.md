# rest-generator

rest-generator for cyan

## Installation

```bash
npm install cyan-rest-generator --save-dev
```

## Usage

### As a CLI tool

```bash
rest-generator <input-path> <output-path>
```

## Features

- Generates maven-rest interfaces (rest, flax, url)
